Welcome to FrOW

FrOW ( Fruits on the Web ) is a fun and interactive game that helps you keep your mind sharp and focused. The game is simple, but it will challenge your memory and attention. You just have to match the cards with the same fruit, but be careful, after each round the cards will shuffle again and you will have less time to pass to the next round. Click the button below to start playing. Good luck!
