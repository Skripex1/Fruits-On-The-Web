Introduction

1.1 Abstract

FrOW (Fruits On The Web) is a project aimed at creating an educational game based on images of fruits and vegetables, intended for elementary school students. The purpose of this game is to provide an interactive and enjoyable way of learning about various types of fruits and vegetables, thereby promoting healthy eating habits.

1.2 Purpose

The purpose of this document is to specify the software requirements for developing the user interface for the educational web game 'FrOW (Fruits On The Web)'. This document exclusively covers the requirements for the user interface, including structure, appearance, interaction, and other relevant aspects and does not include requirements for other components or subsystems of the game. Those will be later implemented and documented here.

1.3 Document Conventions

The text in this document is formatted according to the typography and styling standards recommended within the scholarly HTML format. Alignment, font size, and style ensure a consistent and easily readable presentation.

1.4 Intended Audience

This document is intended for both the team involved in the 'FrOW (Fruits On The Web)' project and individuals outside the team who are interested in the project documentation.

1.5 Product Scope

The purpose of this application is to provide a visually engaging and intuitive platform for elementary school students to learn about different types of fruits and vegetables in an interactive manner. The key objectives and goals of the software include:

  • Creating a user-friendly interface that is accessible to children in primary grades.
  • Incorporating visually appealing graphics and animations to enhance the learning experience.
  • Promoting healthy eating habits and nutrition awareness among young learners.

Overall Description

2.1 Product Perspective

The 'FrOW (Fruits On The Web)' project is conceived as a new product and is not an extension of any existing products within the context of elementary school education.

2.2 Product Functions

For the 'FrOW (Fruits On The Web)' project, the major functions of the front-end include:

  • Displaying the main menu where users can select different games or activities.
  • Providing an interface for users to navigate through various sections of the website.
  • Implementing a leaderboard to display top players scores.
  • Allowing users to interact with elements such as buttons, menus, and forms smoothly.
  • Ensuring responsive design to support different devices and screen sizes.

2.3 Operating Environment

The web application 'FrOW (Fruits On The Web)' can be accessed and used on any operating system (Windows, macOS, Linux) and on any modern web browser, including Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari. There are no special hardware requirements for using the application, and it can be accessed from any device with internet access, including desktops, laptops, tablets, and mobile devices.

2.4 User Documentation

The user documentation for 'FrOW (Fruits On The Web)' is the scholarly HTML document accompanying the application. It serves as a comprehensive guide for users, providing detailed instructions on navigation, features, and troubleshooting. Additionally, a dedicated help page is available within the application interface to assist users with navigating the site.

System Features

3.1 Game Page

The game page will feature a small pop-up allowing users to input their username, which will be later used in the leaderboard table. After entering their username, the actual game will start. Please note that the game itself will be implemented later along with the backend.

3.2 Contact Page

The contact page features a simple form where users can provide their name, email address, and a message or description of their issue. This allows them to submit their questions, suggestions, or problems to the project team or application administrators.

3.3 Leaderboard Page

The Leaderboard page will display rankings of users based on their performance in the game. Users will be able to track their own ranking as well as that of other users, which will help increase engagement.

3.4 About Page

On the "About" page, that you can access by a click on the game logo, you will find essential information about our game and its rules. Here, we will detail the concept behind the game, our objectives, and the rules players need to follow. This page is already here, but the information is not yet displayed as the game logic is not yet implemented.
